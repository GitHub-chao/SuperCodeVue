# SuperCodeVue是什么？

`SuperCodeVue`，是`VUE`+`Element-ui`搭建的一个通用后台管理端模板，包含功能有：  
##### 基础设置:  
    1.人员管理  
    2.角色管理  
    3.部门管理
    ...
##### 系统设置：
    1.系统菜单
    2.角色授权
    3.在线人员
    4.在线客服【webIM功能ok，体验待优化】
    ...

##### 基本满足后台管理的通用模块
